# Modelos de Negócio – AGRODIGITAL

Este documento define os principais tipos de usuários e regras
do marketplace AGRODIGITAL LATAM.

## 1. Produtor (CEASA)
- Vende produtos agrícolas
- Pode operar via box físico ou box virtual
- Define preços e disponibilidade
- Recebe pagamentos com desconto de comissão

## 2. Comprador
- Supermercados
- Restaurantes
- Hortifrútis
- Compradores B2B
- Faz pedidos, agenda entregas ou retiradas

## 3. Logística / Frete
- Transportadores parceiros
- Frete interno CEASA ou externo
- Integração futura com cálculo automático de frete

## 4. Box Virtual
- Espaço digital para produtores sem box físico
- Estoque controlado pela plataforma
- Permite operação 100% digital

## 5. Comissão
- Comissão padrão da plataforma: 8%
- Aplicada sobre pedidos finalizados

## 6. Caixa Virtual
- Registro de entradas e saídas
- Controle de valores a receber
- Base para antecipação de recebíveis

## 7. Antecipação de Recebíveis
- Produtor pode antecipar valores
- Sujeito a regras financeiras e parceiros
- Funcionalidade premium

## 8. Admin
- Gestão de usuários
- Gestão de pedidos
- Controle financeiro
- Relatórios e compliance
